# Code Dojo Tic-Tak-Toe

## Clone
```
git clone https://github.com/gruffT/doJoToe.git
cd doJoToe
```
## Init
``yarn install``

## Run
``yarn start:dev``
Then point your browser to http://localhost:1234

## Test
``yarn test``

## Build
``yarn build``
Outputs to ./dist
